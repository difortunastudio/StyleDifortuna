# Style Difortuna - Web 2026

## Migración de Difortuna Studio a Style Difortuna

Esta carpeta contiene la web actualizada con el nuevo branding de **Style Difortuna**.

### Cambios realizados:

1. **Branding actualizado:**
   - Difortuna Studio → **Style Difortuna**
   - Logo y títulos actualizados en todas las páginas

2. **Email actualizado:**
   - hola@difortunastudio.com → **info@styledifortuna.com**

3. **Dominio actualizado:**
   - CNAME: difortunastudio.com → **styledifortuna.com**

### Archivos incluidos:

- `index.html` - Página principal
- `servicios.html` - Página de servicios
- `difor.html` - Página de colección DIFOR
- `contacto.html` - Página de contacto
- `gracias.html` - Página de agradecimiento
- `css/styles.css` - Estilos principales
- `images/` - Carpeta para imágenes
- `favicon.png` - Icono del sitio
- `CNAME` - Configuración del dominio

### Para subir al hosting:

1. Obtener acceso al dominio `styledifortuna.com`
2. Subir todos los archivos al hosting
3. Configurar el dominio para que apunte a los archivos

### Notas:

- La web mantiene todo el diseño y contenido original de difortunastudio.com
- Solo se han cambiado los elementos de branding especificados
- Las imágenes y fotos actuales se mantienen intactas
